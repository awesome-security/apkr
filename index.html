Features

Upload your Android applications to apkr for free and have them scanned for malware automatically! The results are presented in a user-friendly report.

Free

Easy. What you see is what you get. No fees, no subscritions, nothing.

No cuts

Provide extensive and very detailed information about app analysis result.

High performance

Internal algorithms are exclusively design for bypassing anti-analysis techniques.

String classification

Detect and classify all available strings in different sets.

Resource analysis

Resource decoder, file enumeration, hashing and fuzzing.

Context analysis

Certificate dump, manifest dump, permission risk evaluation etc.

Bytecode analysis

Explore internal assembly instruction of the app.

Call flow graph

Detailed CFG generation based on internal app call tracing.

Behaviour based rules

Behaviour based detection for malware analysis for each submitted app.

Machine learning prediction

Artificial intelligence algorithms are applied to detect unknown or zero day malware apps.

Parameter bypassing

Internal analysis engine uses a innovative VM supervisor that monitors and tracks variables and methods.

Hot swap scalability

Add new features easily using plugin API without modifying core engine nor stopping the service.

Risk score calculation

Simple risk score is calculated for non expert users.

Fully Responsive

Information is shown in a adaptative design for multiple screen sizes.

opcode frequency, debug certificate detection, dex header dump, etc.
